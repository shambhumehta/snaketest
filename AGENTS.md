# Snake

Single-file pygame snake game at `snake.py`.

## Run

```bash
python3 snake.py
```

## Dependencies

- `pygame` — install via `pip3 install pygame` if missing.

## Controls

- Arrow keys to move
- SPACE to restart after game over
- ESC to quit

## Notes

- Game runs at 10 FPS (clock.tick(10) in the main loop).
- No tests, no build system, no linters.
